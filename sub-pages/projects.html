Flask User Verification

This project prompts the user their name and birthday. If a valid birthday is given, then the user recieves a personalized welcome, and is placed in an age group! This age group is determined using an ML model. To reduce redundancy, the model training is stored. If the birthday is not valid, then the user is prompted to try again. With a succesfull operation of the interface, the user can replay as many times as they want.
